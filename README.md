# Scripts em bash
Esse repositório contempla todos os scripts desenvolvidos para o estudo da linguagem, abaixo vai uma descrição de cada um.

## Maneira geral de utilizar cada um deles
1. Clone o repositório em sua máquina:
```bash
git clone git@github.com:leonardorscarpitta/scripts-bash.git
```
2. De a permissão para que você possa executar os scripts sem necessitar de permissão de super usuário `sudo`:
```bash
chmod +x -R scripts-bash
```
3. Rode o script desejado utilizando `./scripts-bash/nome-do-script.sh`;

## Descrição dos Scripts
- `pc-stats.sh`: Com esse script você pode visualizar de uma forma simplificada os recursos que estão sendo consumidos na sua máquina.
